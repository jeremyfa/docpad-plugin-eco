# ECT Plugin for DocPad

<!-- /BADGES -->


Adds support for the [ECT](http://ectjs.com/) (Fastest JavaScript template engine with embedded CoffeeScript syntax) templating engine to [DocPad](https://docpad.org)

Convention:  `.anything.ect`

## Install

```
docpad install ect
```
